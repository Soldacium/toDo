What does this app do?

Well, it it pretty simple, this app is made to store your notes and thoughts right here, on your own browser.

What exacly are it's functions?

You can make notes, toDos, look what you need to do each day of the week, modify anything you want and save. All without logging in!

How does it do it?

The concept is pretty easy. After I've learned about WHOLE 5MB OF LOCAL STORAGE available to each website on the internet, I knew what exacly it's pourpose could be.

5MB = 1024 * 1024 * 5 bytes = 5,242,880 charactes that can be saved on your browser.It doesn't seem like much, but if we consider that avg. book can have ~500,000 characters, and avg. person doesn't write books and esseys in app meant for notes and ToDos, than those ~5,000,000 characters will certainly suffice.

For those programming, I guess It's about as complicated as you'd expect with doing same thing using external database. Maybe less. Probably less.

In service, setup:

            localStorage = localStorage;

            keys = {
                settings: 'sett',
                numOfNotes: 'NN',
                notes: 'NO',
                toDos: 'TD',
            
            }

Then some functions like this:

            saveSettings(settings: object) {
                const settingsString = JSON.stringify(settings);
            
                localStorage.setItem(this.keys.settings,settingsString);
            }
            
            getSettings() {
                return JSON.parse(localStorage.getItem(this.keys.settings));
            }

And finally

            import { LocalDataService } from '../services/local-data.service';

            [...]

            constructor(private localData: LocalDataService) {
                this.settings = localData.getSettings()
            }

That's the main way I did it, all code available on github.

Is it secure?

It's secure from hackers attacking data server, as there is no data server to begin with. It is as secure as a note you could write on your computer's desktop.

It comes at cost at the moment though; your data won't prevail when changing devices, unless I implement logging in, in which case you could, but then,everything less secure and private.
